# Sales-Data-Analytics-with-SQL-and-Python-ClassicModels-Database
End-to-end sales data analysis project combining SQL queries, Python analysis, and business visualizations using the ClassicModels dataset.

## Project Overview
This project explores the ClassicModels database to generate business insights for a miniature vehicle retailer.

## Objective
The goal is to analyze customers, orders, products, sales performance, and employee contributions using SQL and Python.

## Tools
- MySQL
- MySQL Workbench
- Python
- Pandas
- Matplotlib
- GitHub

## Project Structure
- `sql/`: SQL queries for exploratory and business analysis
- `notebooks/`: Python notebooks for visualizations
- `data/`: exported data if needed
- `visuals/`: charts and figures

## SQL Analysis

- 01_schema_introspection.sql: Database structure analysis  
- 02_eda_exploration.sql: Business insights and exploratory analysis
